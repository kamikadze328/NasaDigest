# Nasa Digest
Mobile app project with [Nasa API](https://api.nasa.gov/)

#### Technologies
- Clean Architecture
- MVVM
- Room
- Ktor
- Coroutines + Flow
- Hilt
- Jetpack Navigation
- Jetpack Compose
- Matetial 3
