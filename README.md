# Nasa Digest
Mobile app project with [Nasa API](https://api.nasa.gov/)

#### Technologies
- Clean Architecture + MVVM
- Jetpack Navigation
- Coroutines + Flow
- Matetial 3
- Room
- Ktor
- Hilt
- Jetpack Compose
